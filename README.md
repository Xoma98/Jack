# Гордимся Лапиным Евгением

Статический сайт-портрет о Евгении Лапине: защитнике Родины, друге и семьянине. Две страницы, без фреймворков — чистый HTML/CSS/JS.

## Быстрый старт

```bash
npm install
npm run dev
```

Vite поднимет dev-сервер (обычно `http://localhost:5173/`; если порт занят — следующий свободный).

Другие команды:

```bash
npm run build    # сборка в dist/ (для Vite, в проде не используется)
npm run preview  # превью сборки
```

```css

background: url('../images/evgeny-1.png') center/cover no-repeat;
```

## Как редактировать контент

- **Тексты и карточки** — напрямую в `index.html` и `gallery.html`
- **Стили** — `css/style.css` (переменные в `:root`, шрифт Inter через Google Fonts)
- **JS** — только мобильное меню в `js/main.js`
- **Навигация** — одинаковый header/footer на обеих страницах; при добавлении страницы — продублировать header/footer

Шаблон карточки в галерее:

```html
<article class="card">
  <img class="card-image" src="images/evgeny-N.png" alt="...">
  <div class="card-body">
    <span class="card-tag">Тег</span>
    <h3>Заголовок</h3>
    <p>«Описание про семью или дружбу»</p>
  </div>
</article>
```

## Деплой

Продакшен — **Docker (multi-stage) + Vite build + nginx**.

```bash
docker build -t lapin-site .
docker run -p 8080:80 lapin-site
```

Сборка в два этапа:
1. `npm run build` — Vite собирает `index.html`, `gallery.html`, CSS, JS и все 8 фото в `dist/`
2. nginx отдаёт содержимое `dist/`

- `amvera.yml` — конфиг для [Amvera](https://amvera.ru): сборка из Dockerfile, порт 80
- После деплоя на Amvera: **пересобрать проект** (не просто рестарт), иначе Docker может использовать старый кэш слоёв

### Почему мог показываться «старый» дизайн

1. **Vite не собирал `gallery.html`** — в `dist/` попадала только главная (исправлено в `vite.config.js`)
2. **nginx кэшировал CSS/фото 7 дней с `immutable`** при тех же именах файлов — браузер не подтягивал обновления (исправлено в `nginx.conf`)
3. **Docker копировал исходники напрямую**, минуя `npm run build` — локальный dev и прод могли расходиться (исправлено в `Dockerfile`)

После деплоя обновите страницу с очисткой кэша: `Ctrl+Shift+R`.

## Заметки для AI-ассистента

1. **Язык сайта** — русский. Контент патриотический, про защитника, семью и дружбу.
2. **Нет React/Vue/TS** — не добавлять фреймворки без явной просьбы.
3. **Фото лежат в `images/`**, не в `assets/` (assets — временное хранилище Cursor при загрузке из чата).
4. **Две HTML-страницы** — изменения в навигации/футере нужно синхронизировать в обоих файлах.
5. **Галерея** — 8 карточек; на главной превью — 3 (`evgeny-3`, `evgeny-2`, `evgeny-1`).
6. **Сборка** — `npm run build` через Vite; Docker использует результат сборки (`dist/`), не исходники напрямую.
7. **Порты dev-сервера** — 5173 по умолчанию; при занятости Vite перебирает 5174, 5175 и т.д.
